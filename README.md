# Currency-Identification-using-colour-sensor-ESP32-
This project uses an Arduino, color sensor, and LCD to detect Indian currency notes like ₹10, ₹20, ₹50, ₹100, and ₹500 based on RGB values.  The sensor reads the note’s color, the Arduino compares it with stored values, and the detected currency is shown on the LCD. Accuracy depends on lighting and calibration.
